# Solana Voting Smart Contract

A simple decentralized voting system built on the Solana blockchain using the Anchor framework.  
This project demonstrates how to create, deploy, and interact with a smart contract that allows users to vote between two options and store the results on-chain.

---

## 📌 Project Overview

This project implements a basic voting system where users can:

- Initialize a poll
- Vote for Option A
- Vote for Option B
- View voting results stored on the blockchain

The goal of this project is to demonstrate core concepts of Solana smart contract development using Anchor.

---

## 🧱 Technologies Used

- Solana Blockchain
- Anchor Framework
- Rust
- TypeScript
- Solana Playground
